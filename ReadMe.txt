Apilikacija za D&D

Ime: Tavern Board

Mogućnosti: 
- [ ] DM kreira kampanju
	- [ ] DM može dodati mapu na kojoj su označene bitne lokacije s bitnim likovima
	- [ ] Mapa lokacija ima takvu strukturu da je u nju moguće dodati druge "podmape" npr. ako je na mapi kontinent moguće je kliknuti na neki grad i na njemu dodati mapu grada i tako dalje
	- [ ] U lokacije more dodavati NPC
	- [ ] Generiranje NPC charactera
- [ ] Igrači popunjavaju prazni character sheet 
	- [ ] Mogućnost spremanja character sheeta kao PDF i kao neki format koji kasnije mogu updatati i uploadati
	- [ ] DM ima pristup char sheetovima
	- [ ] Igrači pripremaju spellove u aplikaciju
	- [ ] Igrači mogu upisivati notese u aplikaciju
- [ ] Initiative tracker
	- [ ] Upisivanje inicijativa, automatski poredane
	- [ ] DM dodaje protivnike 
	- [ ] Mogućnost dodavanja opisa i statsa enemya
	- [ ] U bazi stoje .png od enemya i igrača koje je moguće dodati na mapu i upravljati s njima
- [ ] Battle maps
	- [ ] Moguće je ubaciti valstite mape 
	- [ ] Ako je potrebno preko mape je moguće staviti grid
	- [ ] Klikom na .png igrača ili enemya moguće ga je kontrolirati 
		- [ ] igrači mogu kontrolirati svoje likove 
		- [ ] DM može kontrolirati sve likove
